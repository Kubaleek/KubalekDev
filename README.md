## 💻 Instalacja
```bash
- git clone https://github.com/Kubaleek/KubalekDev.git
- npm install
- npm run dev
```
Otwórz http://localhost:3000 w swojej przeglądarce
Aby zbudować aplikację w celu wdrożenia w środowisku produkcyjnym.
```bash
- npm run build
- npm start
```
